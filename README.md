# Excel-Sales-Dashboard
Excel Sales Analysis Dashboard Project

## Project Overview
This project is an Excel-based Sales Analysis Dashboard created to analyze annual sales performance of a retail store.

## Key Features
- Total Sales & Profit Analysis
- Monthly Sales Trend
- Category-wise & Region-wise Performance
- Interactive Dashboard using Pivot Tables & Charts

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning & Formatting
